【工业园区 · Phase 3 产业链与科技】

运行：npm run dev → http://localhost:5173/ （Ctrl+F5 强刷）

【推荐产业链】
  1. 煤矿 → 产出煤炭
  2. 火电厂 → 耗煤发电（钢铁厂需要电力）
  3. 钢铁厂 → 耗电，高收入高污染
  4. 光伏 / 回收 / 绿色中心 → 绿分与减排

【资源短缺】
  煤炭或电力不足时，相关建筑效率下降（顶栏红色提示）

【建筑升级】
  左侧「运营建筑」可花资金升至 Lv.2 / Lv.3

【科技树】
  能源 / 环保 / 工业 三类，消耗资金+绿色积分研究

【仍保留 Phase 2】
  年度排放上限、超排罚款、城市评级 D~S、碳抵消与环保升级
